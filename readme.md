<h3 align="left">Objectives:</h3>
- This repo contains a code for discount spinner for eCommerce store. It loads after a few seconds once user visit the website. This can interated into Shopify, WordPress, Or with any custom storefront


<img width="1544" alt="Spinner View" src="https://user-images.githubusercontent.com/20565907/220866323-27e9822e-fe4a-4ab6-81d0-0a84433eb115.png">

<h3 align="left">Features:</h3>
- Setting up custom offer
- Collectin user email in row data. It can use to integrate any third party like klaviyo, Mailchimp etc...
- Control over spinning speed
- One click option to apply offer
